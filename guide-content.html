시스템 사용 방법

프로젝트 관리

프로젝트를 생성하고, 수정하고, 삭제하는 방법에 대한 가이드입니다.

  • 사이드바에서 '프로젝트' 메뉴를 선택합니다.
  • '+ New Project' 버튼을 클릭하여 새 프로젝트를 생성합니다.
  • 프로젝트 이름, 마감일, 상태, 설명을 입력합니다.
  • 편집 또는 삭제 버튼으로 기존 프로젝트를 관리합니다.

마일스톤 관리

프로젝트 마일스톤을 추가하고 관리하는 방법에 대한 가이드입니다.

  • 사이드바에서 '마일스톤' 메뉴를 선택합니다.
  • 프로젝트 드롭다운에서 관련 프로젝트를 선택합니다.
  • '+ New Milestone' 버튼을 클릭하여 새 마일스톤을 추가합니다.
  • 마일스톤 이름, 날짜, 상태, 설명을 입력합니다.
  • 삭제 버튼으로 기존 마일스톤을 제거할 수 있습니다.

산출물 관리

프로젝트 산출물을 추가하고 관리하는 방법에 대한 가이드입니다.

  • 사이드바에서 '산출물' 메뉴를 선택합니다.
  • 프로젝트 드롭다운에서 관련 프로젝트를 선택합니다.
  • '+ New Deliverable' 버튼을 클릭하여 새 산출물을 추가합니다.
  • 산출물 이름, 날짜, 상태, 설명을 입력합니다.
  • 삭제 버튼으로 기존 산출물을 제거할 수 있습니다.

태스크 관리

프로젝트 태스크를 추가하고 관리하는 방법에 대한 가이드입니다.

  • 사이드바에서 '태스크' 메뉴를 선택합니다.
  • 프로젝트 드롭다운에서 관련 프로젝트를 선택합니다.
  • '+ New Task' 버튼을 클릭하여 새 태스크를 추가합니다.
  • 태스크 이름, 마감일, 상태, 담당자를 입력합니다.
  • 편집 또는 삭제 버튼으로 기존 태스크를 관리합니다.

자주 묻는 질문

데이터는 어디에 저장되나요?

현재 데이터는 브라우저의 localStorage에 저장됩니다. 브라우저 캐시를 지우면 데이터가 삭제될 수 있으니 주의하세요.

여러 사용자가 동시에 작업할 수 있나요?

현재 버전에서는 동시 작업이 지원되지 않습니다. 향후 버전에서 서버 기반 데이터 저장 및 실시간 협업 기능이 추가될 예정입니다.

데이터를 내보내거나 가져올 수 있나요?

현재 버전에서는 데이터 내보내기/가져오기 기능이 지원되지 않습니다. 향후 업데이트에서 CSV 및 JSON 형식의 데이터 내보내기/가져오기 기능이 추가될 예정입니다.

모바일에서도 사용할 수 있나요?

네, 이 시스템은 반응형으로 설계되어 모바일 환경에서도 사용할 수 있습니다. 작은 화면에서는 사이드바가 숨겨지며, 헤더의 메뉴 아이콘을 통해 접근할 수 있습니다.

추가 문의사항이 있으신가요? support@projectmanager.com으로 문의해주세요.
